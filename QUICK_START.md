# Zone Scheduling Quick Start

## What This Site Is For

Zone Scheduling is a live calendar that helps you:

- see school events on the calendar
- plan photographer coverage by zone
- track shared notes
- check whether an event is confirmed

The event data comes from the Google Sheet. Limits, notes, and history are shared through Apps Script.

## Main Views

### Schools

Use this view to:

- see which schools are scheduled on each day
- click a school for event details
- click a day to see all schools on that day

### Zone Staffing

Use this view to:

- see total photographers scheduled each day
- see photographers scheduled by zone
- enter daily and zone limits
- click a day to see which schools make up the staffing totals

## What The Colors Mean

- Yellow: within 2 below the limit
- Red: at the limit
- Dark red: over the limit

Weekends are light purple.

## Notes

- Click the day number to add or view a note for that day
- In `Zone Staffing`, click a zone’s `Note` button to add a note for that specific zone
- Notes are shared across computers

## Filters

You can filter by:

- Zone
- Star Ranking
- Event Type
- Sent
- Confirmed

When filters are active, the calendar shows a `Filtered View` banner above the calendar.

## School Search

Use `School Search` to:

- find a school quickly
- see all of its dates
- review its moved-date history

Clicking a search result jumps directly to that day in `Zone Staffing`.

## Confirmed

The `Confirmed` attribute filter is based on the `Confirmed` field pulled from the sheet. It helps track what has already been added to Roosted.

## Typical Workflow

1. Open `Zone Staffing`
2. Jump to the current month
3. Review daily totals and zone totals
4. Enter or adjust limits
5. Look for yellow, red, and dark red warnings
6. Add notes when the team needs context
7. Use `Confirmed` to filter what has already been added to Roosted
8. Use `School Search` to check a specific school
